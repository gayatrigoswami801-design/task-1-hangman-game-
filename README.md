# 🎯 Hangman Game

## 📖 Overview
Hangman Game is a fun command-line word guessing game built using Python. The player must guess the hidden word one letter at a time before running out of attempts. This project demonstrates core Python concepts such as loops, conditionals, functions, strings, and lists.

---

## ✨ Features
- Random word selection
- Interactive command-line gameplay
- Limited attempts/lives system
- Tracks correct and incorrect guesses
- Win and lose conditions
- User-friendly interface
- Replay support (if implemented)

## 🎮 How to Play

1. Start the game.
2. A hidden word will be selected randomly.
3. Enter one letter at a time.
4. Correct guesses reveal letters in the word.
5. Incorrect guesses reduce your remaining lives.
6. Guess the entire word before your lives run out.

---

## 📸 Sample Gameplay

```text
Welcome to Hangman!
```
